# Click Assistant 移动端目录

本文档用于说明 Click Assistant 移动端相关工程和资料。

当前移动端处于 Android-first 可行性原型阶段，优先验证辅助功能授权、固定坐标点击、停止入口和本地任务保存。

## 目录

- `android/`：Android 可行性原型工程。
