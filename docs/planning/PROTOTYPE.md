# Click Assistant 低保真原型蓝图

文档版本：v0.1.0
最后更新：2026-07-09
维护人：项目负责人
当前状态：初稿，待确认

本文档用于记录 Click Assistant 的低保真页面蓝图。页面设计以“任务配置清晰、执行控制醒目、运行状态可追踪”为核心。

## 1. 设计原则

- 第一屏直接进入工具主界面，不做营销式首页。
- 界面以高频操作为中心：选择任务、编辑步骤、启动或停止执行。
- 运行控制区必须醒目，停止操作必须容易找到。
- 信息密度适中，优先保证重复使用时的效率。
- 默认中文界面，专业术语可保留英文并补充中文含义。

## 3. 新建或编辑任务

新建和编辑任务可以使用同一套表单。

```text
┌──────────────────────────── 新建任务 ────────────────────────────┐
│ 任务名称        [                                      ]          │
│ 任务描述        [                                      ]          │
│ 重复次数        [ 10 ]        开始延迟        [ 3 秒 ]             │
│                                                                  │
│ 默认步骤间隔    [ 500 毫秒 ]  完成后行为      [ 停止任务 ▼ ]       │
│                                                                  │
│ 快捷键设置                                                      │
│ 启动/暂停       [ Ctrl + Alt + S ]                               │
│ 立即停止        [ Ctrl + Alt + X ]                               │
│                                                                  │
│                                      [取消] [保存]                │
└──────────────────────────────────────────────────────────────────┘
```

## 4. 输入步骤编辑

输入步骤编辑用于配置单个鼠标、键盘、组合键或文本输入动作。

```text
┌─────────────────────────── 编辑输入步骤 ─────────────────────────┐
│ 步骤名称        [ 点击确认按钮                         ]          │
│ 是否启用        [x] 启用                                          │
│ 坐标位置        X [ 120 ]    Y [ 360 ]    [采集坐标]               │
│ 点击类型        [ 左键单击 ▼ ]                                    │
│ 执行前等待      [ 0 毫秒 ]                                        │
│ 执行后间隔      [ 500 毫秒 ]                                      │
│                                                                  │
│                                      [取消] [保存步骤]            │
└──────────────────────────────────────────────────────────────────┘
```

## 8. MVP 页面清单

- 主界面：任务列表、任务详情、运行状态和最近日志。
- 新建或编辑任务弹窗：配置任务基础信息和快捷键。
- 输入步骤编辑弹窗：配置动作类型、坐标、点击类型、键盘按键、组合键、文本内容和间隔。
- 坐标采集模式：采集当前鼠标位置。
- 设置页面：配置默认间隔、倒计时和快捷键。
